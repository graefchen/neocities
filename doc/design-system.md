# design system

> [!important]
> The design system should be very simple and very clear. People should understand it without problems when interacting with it. Additionally it shall be designed to be fairly modular, to take parts of it, to mix it, and generally have fun using and working with it.
> The css shall be written in the [BEM](https://getbem.com) methodology.

## font

The default font _should be_ sans-serif, the fall-back for $$font-name$$ should be `system-ui, sans-serif`.

## colors

The color-palette includes the colors:

- #121212
- #ffffff
- #e45858
- #999999
